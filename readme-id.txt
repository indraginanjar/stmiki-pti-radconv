Keterangan:

Radix Converter

Aplikasi command line windows (DOS console untuk melakukan konversi antar radix pada 
sebuah bilangan.


Keterbatasan yang diketahui:

Hanya mengolah bilangan bulat positif
Bilangan yang mampu diolah paling banyak dari dan ke 14 digit radix desimal


Untuk menjalankan Radix Converter:

Setelah berada di folder radconv masuk ke folder bin\ didalamnya terdapat satu
berkas exe bernama "radconv.exe". Berkas inilah yang disebut Radix Converter.
Berkas ini bisa disalin atau pun dipindahkan ke folder manapun, karena tetap akan 
berfungsi dengan baik.
